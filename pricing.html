Our pricing is simple and transparent. All customers receive all our features and services, and we charge a fee proportional to the size of the application, summed over three dimensions:

Per vcpu hour: $0.09
Per memory Gb hour: $0.009
Per disk Gb hour: $0.0003

Included is everything you need to run a production service with confidence:

24/7 operations
Support
Continuous deployment
Automatic Vespa/OS upgrades
Strong security
Autoscaling
Performance tune-up program eligibility
Development cloud

Contact us if you want priority support, committed resources at lower cost, or have other pricing requests.

The fine print

Resources allocated in production zones are precisely what specified in services.xml for the duration the application is deployed. If resources are specified in ranges (autoscaling) the resources allocated at any time is the optimal allocation inside the range. In dev and test zones, applications will by default be downscaled, and charges are by the actual usage summed over time.

In addition, we reserve the right to pass on network ingress/egress costs incurred by application usage at standard AWS prices.

Free Trial

You can start a free trial for 14 days at any time. No credit card required. This allows you to add as many users and applications as you like under your tenant for free, up to some max total spend per hour. Contact us if you need additional capacity for trying your use case.
